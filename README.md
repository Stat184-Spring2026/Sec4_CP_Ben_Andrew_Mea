# STAT184-Final Project-Ben-Andrew-Mea

In this project we will be using data from Carnegie Mellon University surrounding the effect of nightly sleep, race, gender, and first generation status on GPA in first year college students. We will have to wrangle and tidy the data and create effective data visualizations to show any relationships within the data we are observing using the ggplot library in R. We will include multiple data visualizations and write in depth summaries to help users better understand what the data is saying

3.) filter() out null values.

5.) pivot_longer() to reduce the number of cols.

7.) successful wrangle and tidy the dataset of our choice
